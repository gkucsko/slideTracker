Check slideTracker Status

Use to check if your add-in is working correctly (ie. needs update).
POST /status
Parameters
Name Type Description
version string Your add-in version number
Response
	Status: 200 OK
	[
		{	
			error: 0,
			message: "you have the newest version running"
		}
	]

Create Presentations

Use to create a new presentation entry.
POST /api/v1/presentations
Parameters
Name Type Description
key string API key to verify you are not a bot.
creator string Name of creator of presentation. Should be the MAC address.
n_slides number Total number of slides in presentation.
Response
	Status: 201 OK

			presentation: {

Update Presentations

Use to update information about the specified presentation.
PUT /api/v1/presentations/<pres_ID>
Parameters
Name Type Description
key string API key to verify you are not a bot.
passHash string Password you received when creating the presentation to verify ownership.
n_slides number Total number of slides in the presentation.
cur_slide number Currently active slide.
active boolean Indicate if presentation is ready for broadcasting.
Response
	Status: 201 OK
	[
		{	
			"pres_ID": efa12,
			"creator": F04DA23A82F4,
			"n_slides": 5,
			"cur_slide": 1,
			"active": true,
			"download": true,
			"file_size": 0,
			"clients": 0,
			"created": 2015-01-13T01:31:21.829Z,
			"updated": 2015-01-13T01:31:21.829Z
		}
	]

Delete Presentations

Use to delete specified presentation and clean up.
PUT /api/v1/presentations/<pres_ID>/delete
Parameters
Name Type Description
key string API key to verify you are not a bot.
passHash string Password you received when creating the presentation to verify ownership.
Response
	Status: 200 OK

Upload Slides

Use to upload slides to the specified presentation.
POST /api/v1/presentations/<pres_ID>/slides
Parameters
Name Type Description
key string API key to verify you are not a bot.
passHash string Password you received when creating the presentation to verify ownership.
slide_ID number Number of uploaded slide, ie. 1-24.
slide file Slide file in PNG format. Mimetype has to be image/png
Response
	Status: 201 OK
	[
		{	
			"pres_ID": efa12,
			"creator": F04DA23A82F4,
			"n_slides": 5,
			"cur_slide": 1,
			"active": true,
			"download": true,
			"file_size": 0,
			"clients": 0,
			"created": 2015-01-13T01:31:21.829Z,
			"updated": 2015-01-13T01:31:21.829Z
		}
	]
