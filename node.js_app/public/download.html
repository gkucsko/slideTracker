SlideTracker is a work in progress. We are constantly adding new features and improving stability. To see the latest progress or contact us for bugs or improvements have a look at our github repository.

Installing the PowerPoint Add-in

STEP 1:
Make sure you have Windows and PowerPoint 2010/13 installed on your system. Then download the installer:
Download v-0.1.0.0
STEP 2:
Make sure all MS office programs are closed.
STEP 3:
Unzip the file and run setup.exe. (It might be necessary to click OK to trust the program.)
STEP 4:
Finish the installation and run PowerPoint.
STEP 5:
Verify that you can see the 'slideTracker' tab. Done!
*NOTE:
If you can't see the add-in or have other problems, please contact us.

Using the add-in:

STEP 1:
Go to the slideTracker tab and click 'Start Broadcasting'. Done! You audience can follow the presentation by going to slidetracker.org and typing in the ID that will be automatically shown on your slides.
STEP 2 (optional):
You can go to the options menu and adjust settings such as ID display location or allow PDF download.
*NOTE:
Closing PowerPoint will automatically delete your presentation from our servers. If for some reason the connection is interrupted we will automatically clean up after a few hours of inactivity.
